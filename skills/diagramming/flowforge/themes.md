# Color Themes

6 themes available. Default: **tech-blue**.

## Semantic Color Roles

| Role | Usage |
|------|-------|
| **primary** | Start/end nodes, lane headers, key entry/exit points |
| **process** | Core processing steps (default for most nodes) |
| **accent** | 1-2 key turning points or decision highlights |
| **neutral** | External systems, data stores, background context |
| **success** | Positive outcomes, completed states |
| **warning** | Decision points, pending states |
| **error** | Failures, exceptions, pain points |
| **secondary** | Optional/alternate paths |
| **storage** | Databases, caches, queues |
| **group** | Dashed group boxes |
