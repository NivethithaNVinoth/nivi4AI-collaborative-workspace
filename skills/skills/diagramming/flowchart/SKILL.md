# Flowchart Skill

Produces a self-contained HTML flowchart with:
- Process boxes: 160x50px
- Decision diamonds: 84x84px  
- Data parallelograms: 160x50px (inputs/outputs)
- Start/end ovals: 130x50px
- Yes/No labels on decision branches
- Left-to-right or top-down layout
