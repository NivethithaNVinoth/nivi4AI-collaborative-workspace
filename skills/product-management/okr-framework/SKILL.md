## Purpose
Turn a strategy into measurable quarterly commitments a team can own and track.
Based on John Doerr Measure What Matters + Google Ventures OKR methodology.

## Key concepts
- Objective: aspirational, qualitative direction (3-5 per quarter max).
- Key Result: quantitative measure â€” "Verb metric from baseline to target by date".
- Aspirational OKRs: 70% hit = success. Committed OKRs: 100% expected.
- Health metric: guardrail that must not degrade while chasing KRs.

## Steps
1. Identify 3 strategic priorities for the quarter.
2. Write one Objective per priority (no numbers).
3. Write 2-4 Key Results per Objective.
4. Assign each KR a type (aspirational/committed) and owner.
5. Add 1-2 health metrics per Objective.
6. Create weekly confidence check-in cadence.
